# CSS Grid Layouts

Examples of css grid layout using:

```css
grid-template-columns: repeat(auto-fit, minmax(240px, 1fr));

grid-auto-rows: 240px;

grid-row: span 2 / auto;

grid-column: span 2 / auto;
```
